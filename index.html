KFDtool Firmware Notice

KFDtool must be running firmware v1.4.0 for proper operation

  1. *WHEN PUBLISHED* Download current firmware here, then decompress
  2. *ONLY FOR TESTING* Download beta firmware here, then decompress
  3. Open KFDtool v1.5.0 on a Windows computer
  4. Select Utility -> Adapter Self Test -> Enter BSL Mode
  5. Select Utility -> Initialize Adapter, choose the decompressed cfp file, select the HW version (on the bottom of the KFDtool software, or on the KFDtool label)
